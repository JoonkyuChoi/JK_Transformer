## 사용법

### 콘솔 옵션들
```
e   : [필수:공통] 세대 (훈련=1~, 추론=0)
v   : [필수:공통] 어휘사전 규모 (tiny ~ huge)
o   : [필수:공통] 원언어 (매칭문장/어휘 사전 로드)
t   : [필수:공통] 번역어 (매칭문장/어휘 사전 로드)
m   : [----:공통] 사전훈련모델 이진파일 (훈련=미세조정, 추론=필수모델)

c   : [옵션:훈련] 매칭문장사전 규모 (tiny ~ huge)
f   : [옵션:훈련] 매칭문장사전 파일 (강제 지정용)
d   : [옵션:훈련] 누락율 (default: 0.2)
l   : [옵션:훈련] 학습률 (default: 0.001)
```

### 언어
콘솔 옵션 [o, t]에 지원되는 설정값
```
en  : 영어
ko  : 한글(default)
ja  : 일어
```

### 규모
콘솔 옵션 [v, c]에 지원되는 설정값
```
tiny: 소형(default)
min0: 저형
min1: 저형
min2: 저형
midi: 중형
big0: 거형
big1: 거형
big2: 거형
huge: 대형
```
